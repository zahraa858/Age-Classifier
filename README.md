# AgeClassifier

This is a simple Java console programfor beginners.  

It does the following:

- Asks the user for their **name**
- Asks for their **age**
- Asks if they are a **student** (true/false)
- Prints personalized messages using **if-else statements**
- Classifies the user as **baby, child, adult, or senior**
- Prints whether the user is a **student or not**
